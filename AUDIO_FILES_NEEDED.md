# Audio Files Required

This application requires 8 audio files (MP3 format) to be placed in the root directory:

1. `room1.mp3` - For Room 1: Who I Am & How I See Engineering Design
2. `room2.mp3` - For Room 2: My Beliefs About Design
3. `room3.mp3` - For Room 3: My Values
4. `room4.mp3` - For Room 4: How I Practice Engineering Design
5. `room5.mp3` - For Room 5: Experience 1 – Cellphone Microscope
6. `room6.mp3` - For Room 6: Experience 2 – Being Challenged
7. `room7.mp3` - For Room 7: Reflection – Where I Am Now
8. `room8.mp3` - For Room 8: Conclusion – My Future Direction

Each audio file will automatically play when the user enters the corresponding room using the up/down arrow keys.

The audio files should match the theme and mood of each room as described in the position statement.
